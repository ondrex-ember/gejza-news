O aplikaci
Jouza Radio&News je otevřená a dynamicky se vyvíjející lokální platforma agregující zprávy, rádiové streamy, počasí a smart data regionu Nový Bor a širší okolí. Je vyvíjena na vlastní náklady a s přihlédnutím k potřebám obyvatel regionu.

Popis aplikace

Jouza Radio&News je nezávislý agregátor zpráv a živých rádiových streamů zaměřený na Severočeský region — konkrétně Liberecký a Ústecký kraj. Cílem je nabídnout přehledné, reklamou nezatížené rozhraní pro každodenní přísun informací a hudby.

Aplikace funguje jako jednostránkový klient (SPA) nasazený přes Vercel a verzovaný na GitHubu. Data zpráv jsou stahována z live JSON feedu, počasí obstarává Open-Meteo API doplněné osobní meteorologickou stanicí v Novém Boru. Sekce Smart Region agreguje data ze senzorové sítě — ovzduší, voda, půda, radiace a silniční provoz I/9.

Záložka Příroda přináší data lesnického monitoringu z DendroNet — les Jetřichovice a les Ralsko. Aplikace nevyužívá žádné cookies třetích stran ani placenou inzerci.
